# 1.1.2

### 한국어

```csharp
1.1.2 업데이트

1. 자동 로그인 기능 추가
- 이제 직전에 선택한 로그인 방법으로 자동 로그인을 시도합니다.
- 로그인 방법을 변경하고 싶다면 설정의 로그아웃을 이용해주세요.

2. 미니게임 부활 기능 추가
- 이제 미니게임에서 부활 기회가 주어집니다.
- 부활은 광고 시청 또는 유료 재화 사용으로 할 수 있습니다.

3. 게임 버전 확인
- 이제 설치하지 않은 업데이트가 있으면 팝업으로 알려드립니다.
- 팝업을 닫으면 스토어 링크가 열리지만 게임은 진행할 수 있습니다.
```

### 영어

```csharp
1.1.2 Update

1. Added automatic login function
- Now, automatic login is attempted with the login method selected immediately before.
- If you want to change the login method, please use Logout in Settings.

2. Added mini-game resurrection function
- You are now given a chance to resurrection in the minigame.
- Resurrection can be done by watching advertisements or using paid goods.

3. Check the game version
- Now, if there is an update that has not been installed, a pop-up will notify you.
- Closing the popup will open the store link, but you can proceed with the game.
```

### 한국어

```csharp
1.1.4 업데이트

```

### 영어

```csharp

1. Piyoko growth time reduction function added
- You can now shorten Piyoko's growth time by 1 hour by watching an ad.

2. Added free jelly to the shop
- You can now get free jelly after watching an ad every 3 hours in the shop.

3. Other bug fixes
-Fixed an issue an update guide was displayed even though the app was up to date.
-Fixed a bug that automatic sign-in for Apple sign-in only did not work.
```

# 1.2.3

```csharp
- 게임 오버 시 퍼즐 조작을 못하도록 수정
- 게임 오버 시 피버 발동이 되지 않게 수정
- 부활하였을 때 퍼즐이 섞이지 않는 문제 수정
- 유령이 생성되지 않는 문제 수정
- 콤보 보이스 음소거 기능 추가
- 네코팡 스코어 획득량 낮춤

버그 수정은 이거 두 개가 전부에요
- 부활하였을 때 퍼즐이 섞이지 않는 문제 수정
- 유령이 생성되지 않는 문제 수정
```

### 한국어

```csharp
1.2.3 업데이트

1. 네코팡 버그 수정
- 게임이 종료된 이후 잠시 게임을 진행할 수 있던 문제가 수정되었습니다.
- 부활 기능을 사용한 후에 퍼즐이 섞이지 않던 문제가 수정되었습니다.
- 정상적으로 유령이 생성되지 않던 문제가 수정되었습니다.
- 설정에서 효과음에 대한 버튼을 눌러도 작동하지 않던 문제가 수정되었습니다.
- 비정상적으로 많은 경험치를 획득하던 현상이 수정되었습니다.

2. 설정
- 목소리를 켜고 끌 수 있는 기능이 추가되었습니다.
- 해당 옵션을 통해 네코팡의 콤보 음성을 켜고 끌 수 있습니다.

3. 기타
- 외부 활동에 의한 대기 시간이 기존 2분에서 20분으로 증가되었습니다.
- 예를 들어, 기존에는 게임 중 2분 이상 스마트폰으로 전화를 하면 게임이 자동 종료되었습니다. 하지만 이제는 최대 20분까지 기다려 드립니다.
```

### 영어

```csharp
1.2.3 update

1. Nekopang bug fix
- The problem of being able to play the game for a while after the game ended has been fixed.
- Fixed an issue where puzzles were not mixed after using the resurrection function.
- An issue where ghosts were not spawned normally has been fixed.
- Fixed an issue where pressing the button for sound effects in settings did not work.
- The phenomenon of obtaining an abnormally large amount of EXP has been fixed.

2. Settings
- Added ability to turn voice on/off.
- Through this option, you can turn on/off Neko Pang's combo voice.

3. Other
- The waiting time for external activities has been increased from 2 minutes to 20 minutes.
- For example, in the past, if you make a phone call on your smartphone for more than 2 minutes during a game, the game automatically ends. But now we wait up to 20 minutes.
```
